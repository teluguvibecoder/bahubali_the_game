# Bahubali - The Game

A browser-based physics slingshot game inspired by Angry Birds, featuring Bahubali theme and Telugu dialogues.

## About

Launch soldiers from a coconut tree slingshot and try to get them past the Mahishmati wall. Earn points for successful throws and avoid hitting the wall!

## How to Play

1. **Start**: Click anywhere to start the game (landscape mode recommended)
2. **Aim**: Drag the soldier on the coconut tree to pull back the slingshot
3. **Launch**: Release to fire the soldier
4. **Score**:
   - Pass the wall = +1 point
   - Hit the wall = -1 point
   - Hit the wall 3 times = Game Over
5. **Celebrate**: Get meme celebrations every 5 points!

## Features

- Physics-based slingshot mechanics
- Trajectory prediction while aiming
- Score tracking system
- Sound effects for launches and hits
- Background music
- Meme celebrations at score milestones
- Telugu-themed game over screen
- Mobile and desktop support
- Fullscreen mode

## Tech Stack

- React 18
- Matter.js (Physics Engine)
- Vite
- Howler.js (Audio)
- GSAP (Animations)
